My First Game

This is the first game I have ever made. When I decided to go back to college, I attended an open day at the HKU (March). What drew me to the HKU is that as a developer you collaborate with game art, game design and interaction design. However, to be admitted you had to submit a (digital) game and a portfolio. Also a limited number of students are accepted at the HKU, about 100 people for 4 different studies. The last moment to hand in my game and portfolio was in May. So I had a few months to learn to code and make some kind of game.

As for the game, it is quite simple. The player is attracted to the planet and can move around it. During the time, stars spawn. Try to pick up as much as possible within the match time. In addition, you have a speed boost that you can pick up to go faster and there is a potion bottle that you have to avoid. This project is not the most impressive and the best in quality, but it shows where I started. It also shows the improvement i have made along my journey. 


public float gravity;

public void Attract(Transform body) {
    Vector3 gravityUp = (body.position - transform.position).normalized;
    Vector3 bodyUp = body.up;

    body.GetComponent().AddForce(gravityUp * gravity);

    Quaternion targetRotation = Quaternion.FromToRotation (bodyUp,gravityUp) * body.rotation;
    body.rotation = Quaternion.Slerp (body.rotation,targetRotation, 50 * Time.deltaTime);
    }

void Update () {
    moveDir = new Vector3 (Input.GetAxisRaw ("Horizontal"), 0, Input.GetAxisRaw ("Vertical")).normalized;
}

void FixedUpdate () {
    GetComponent().MovePosition(GetComponent().position + transform.TransformDirection(moveDir) * moveSpeed * Time.deltaTime);
}
