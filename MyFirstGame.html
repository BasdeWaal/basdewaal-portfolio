My First Game

This is the first game I have ever made. When I decided to go back to college, I attended an open day in March 2016 at the HKU. What made me decide to go for HKU is that here the study has a combination with not only design but also with art and interaction design. The three most important elements of a good game at least in my opinion. The admission procedure asks from every new applicant to submit a (digital) game and a portfolio. The reason is that every year only a limited number of students is accepted at the HKU, about 100 for the total of 4 different studies. So being a novice at that moment I only had a few months to hand in my game and portfolio. This game was my first effort and made that I was selected.

As for the game, it is quite simple. The player is attracted to the planet and can move around it. During the time, stars spawn. The goal is to try to pick up as many stars as possible within the match time. In addition, you have a speed boost that you can collect to go faster and there is a bottle with potion that you have to avoid. This project is not the most impressive, but it shows where I started and how I improved during my study over the four years. 


public float gravity;

public void Attract(Transform body) {
    Vector3 gravityUp = (body.position - transform.position).normalized;
    Vector3 bodyUp = body.up;

    body.GetComponent().AddForce(gravityUp * gravity);

    Quaternion targetRotation = Quaternion.FromToRotation (bodyUp,gravityUp) * body.rotation;
    body.rotation = Quaternion.Slerp (body.rotation,targetRotation, 50 * Time.deltaTime);
}

void Update () {
    moveDir = new Vector3 (Input.GetAxisRaw ("Horizontal"), 0, Input.GetAxisRaw ("Vertical")).normalized;
}

void FixedUpdate () {
    GetComponent().MovePosition(GetComponent().position + transform.TransformDirection(moveDir) * moveSpeed * Time.deltaTime);
}
