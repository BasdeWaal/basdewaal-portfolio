Sustainability

As starters for this project we had a 3 day during game-jam. Giving words to the term sustainability. We had to translate that into a game (concept) . This was a group assignment. Goal was to make the spaceship as “green” as possible and include a number of features to recycle it during it’s voyage. How to develop this was my task during this project, together with 2 artists and 1 designer. The ship has to survive as long as possible. So you have to be alert on maintenance, energy and try to upgrade your systems. Because, in the meanwhile, you also have to avoid outside dangers that can make impacts on the structure of the ship and eventually destroy it.

To stay alive in this game, you have to keep reinforcing and rebuilding the ships walls. Because they player is not getting any oxygen when he is in a room with a missing wall. Several scripts need to know if there is a breach in the spaceship. So I thought this was the right moment to learn more about events and how to use them. 


public static event System.Action roomBreachedImpact;

public void RoomBreachedImpact(GameObject breachedRoom){
    if (roomBreachedImpact != null){
        roomBreachedImpact(breachedRoom);
    }
}

    PlayerController.roomBreachedImpact += CheckRoomsForBreaches;

This was also one of the few groups projects that we had a sound designer in our group. So that means that I had the opportunity to play with some sound effects. To make it easy for the sound designer, I made a simple editor script where he can easily fill in the sound effects. 


public static AudioManager instance;

public SoundGroup[] soundGroups;
private Dictionary soundGroupsDictionary = new Dictionary();

private void Awake(){
    if (instance == null){
        instance = this;
    }
    else{
        Destroy(gameObject);
    }
    
    foreach (SoundGroup soundGroup in soundGroups){
        groupDictionary.Add(soundGroup.GroupID, soundGroup.group);
    }
}
                           
public AudioClip GetRandomClipFromName(string name){
    if (soundGroupsDictionary.ContainsKey(name)){
        AudioClip[] sounds = soundGroupsDictionary[name];
        return sounds[Random.Range(0, sounds.Length)];
    }

    return null;
}

public AudioClip GetClipFromName(string name, int index){
    if (soundGroupsDictionary.ContainsKey(name)){
        AudioClip[] sounds = soundGroupsDictionary[name];
        return sounds[index];
    }

    return null;
}

[System.Serializable]
public class SoundGroup{
    public string GroupID;
    public AudioClip[] group;
}
