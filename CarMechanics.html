Car Mechanics

Car Mechanics was a “free” project so I first had to write my own proposition. In this project I made the choice to challenge myself in developing race-mechanics. Goal was that the handling of a car had to feel natural while driving it in an open world. By making some simple “tweaks” it was also possible to change the behavior of the car. In this project I developed 2 cars that felt totally different to move around. Besides cruising around you could drift with the cars and they were equipped with suspension making it more fun to leave to road.

This project was in my freshman year. So Unity was still new for me and I didn't had much experience with programming yet. This project was about trying out several things and reading a lot about the first rules of programming. At the end I found that I had the best results with the Unity build in wheel colliders. With a lot of tweaking I got something that felt like one is really able to controlling the car. 


private void Drifting(bool drift){
    if (drift){
        foreach (WheelCollider WheelC in wc){
            WheelFrictionCurve sf = WheelC.sidewaysFriction;
            sf.stiffness = 1;
            WheelC.sidewaysFriction = sf;
        }
    }
    else {
        WheelFrictionCurve sf0 = wc[0].sidewaysFriction;
        WheelFrictionCurve sf1 = wc[1].sidewaysFriction;
        WheelFrictionCurve sf2 = wc[2].sidewaysFriction;
        WheelFrictionCurve sf3 = wc[3].sidewaysFriction;
        sf0.stiffness = 3;
        sf1.stiffness = 3;
        sf2.stiffness = 4;
        sf3.stiffness = 4;
        wc[0].sidewaysFriction = sf0;
        wc[1].sidewaysFriction = sf1;
        wc[2].sidewaysFriction = sf2;
        wc[3].sidewaysFriction = sf3;
    }
}
